Freqtrades中文快捷命令程序。
一键傻瓜式操作。不管新手还是老鸟都很适用。
一键下载/回测/模拟盘实盘切换（需断开电源）/一键重启/
自选配置/策略/超参数优化类别自选/时间自选。
自定义所有参数然后预览生成的命令最后点击执行。
功能还是很丰富的。
具体功能请大家查看图片。
必须放在Freqtrades量化程序的根目录（必须和docker-compose.yml在一起）。
可右键空白区域打开终端输入python kq.py开启本程序 或者自己打包成EXE程序（推荐！）
重点说下：模拟盘✔   取消✔则为实盘。 实盘！真金白银一定要注意。有二次确认。
币种自行输入/合约输入(代币名称/usdt：usdt)并勾选合约模式。不勾选填(代币名称/USDT).有历史记录可选。
可在左下角一键打开本地网页UI。右键打开根目录。
注意端口。
本程序是为我自己使用开发的。有问题自己去py文件看看代码。
运行有问题可看实验室日志让ai分析。使用问题可以+v 599267295咨询。
